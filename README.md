# High-Pressure Transcritical CO2 Cooling Loop Analysis
Numerical analysis and CAD design of a transcritical CO2 cooling loop for high-pressure detector thermal management, featuring Siemens NX routing and ANSYS Fluent CFD validation.

## Project Overview
This repository contains the engineering schematics, geometric models, and computational fluid dynamics (CFD) simulation files for a high-pressure transcritical $\text{CO}_2$ cooling loop. Designed with modern particle detector thermal management constraints in mind, this project focuses on optimizing routing geometry to mitigate turbulence and validating fluid flow integrity under extreme operating parameters.

----

## Technical Specifications & Features

*   **Fluid Routing Network:** Designed a compact 12 mm fluid routing network within **Siemens NX** using an orthogonal layout.
*   **Turbulence Mitigation:** Implemented a strict **2x OD bend radius** for the cooling fluid pathways to minimize localized pressure drops and fluid turbulence.
*   **CFD Validation:** Executed finite volume fluid flow analyses using **ANSYS Fluent** to validate velocity profiles and verify structural loop integrity under high-pressure parameters.
*   **Thermal Insulation:** Designed targeted 3 mm thermal insulation sleeves on the low-pressure suction lines to block parasitic ambient heat gain and eliminate condensation risks near sensitive electronic
  boundaries.

----

## Tools & Software Used

*   **CAD/Geometric Modeling:** Siemens NX
*   **Simulation & Engineering:** ANSYS Fluent (CFD)
